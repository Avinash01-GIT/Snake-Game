# Snake Game

A classic Snake Game implementation using HTML, CSS, and JavaScript.

## Demo

You can play the game live [here](https://avinash01-git.github.io/Snake-Game/).

## Features

- Enjoy the nostalgic experience of the classic Snake Game.
- Responsive design optimized for laptop and desktop use.
- Simple controls using arrow keys for directional movement.
- Track your score as you progress through the game.

## How to Play

1. Use the arrow keys (up, down, left, right) to control the snake's movement.
2. Guide the snake to eat the food (gray square) to grow in length.
3. Avoid collisions with the walls or the snake's own body.
4. The game ends when the snake collides with a wall or itself.
5. Aim to achieve the highest score possible by growing the snake longer.
